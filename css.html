行间样式使用

今天在弄 css画图形 的时候，要用到伪元素，但是伪元素不能在行内使用(貌似是，反正没有找到方法)，所以去看别怎么弄的画图的，我了个去，居然可以在body内容中<style>标签，一般教程都没有说这样用啊，一般文档都是让style标签放在标签之内。

<body>
<style>
<!--
#contianer{
color:red;
}
-->
</style>
<div id="container"></div>
</body>

CSS画图形

1.正方形

.square{
width:100px;
height:100px;
backgroud:red;}

2.长方形

.rectangle{
width:150px;
height:100px;
background:red;}

3.圆形

.circle {
//将正方形圆角放大到最大
width:100px;
height: 100px;
background:red;
border-top-left-radius:50px;
border-top-right-radius:50px;
border-bottom-left-radius:50px;
border-bottom-right-radius:50px;}

4.椭圆

.oval{
//长方形的圆角放大到最大
width:200px;
height: 100px;
background:red;
border-top-left-radius:100px 50px;
border-top-right-radius:100px 50px;
border-bottom-left-radius:100px 50px;
border-bottom-right-radius:100px 50px;}

5.三角形

.triangle{
//利用边框填充，
width:0;
height:0;
border-bottom:50px solid red;
border-left:25px solid transparent;
border-right:25px solid transparent;}

6.直角三角形

.right-angled-triangle{
//常规三角形的一半，将底边延长即为图例
height:0;
width:0;
border-bottom:50px solid red;
border-right:50px solid transparent;}

7.平行四边形

.parallelogram{
//将长方形沿x或y轴倾斜某个角度即为平行四边形
width:100px;
height:50px;
background:red;
transform: skew(30deg,0);}

8.梯形

.trapezoid{
//对三角形设置宽度
width:100px;
height:0;
border-bottom:100px solid red;
border-left:50px solid transparent;
border-right:50px solid transparent;

}

9.六角星

 

#star-six {
width: 0;
height: 0;
border-left: 50px solid transparent;
border-right: 50px solid transparent;
border-bottom: 100px solid red;
}
#star-six:after {
width: 0;
height: 0;
border-left: 50px solid transparent;
border-right: 50px solid transparent;
border-top: 100px solid red;
content: "";
top: 30px;
left: -50px;
}

10.五角星

 

#star-five{
//画左上与右下的一个三角形
margin:50px 0;
display:block;
position:relative;
color:red;
width:0;
height:0;
border-right:100px solid transparent;
border-bottom:70px solid red;
border-left:100px solid transparent;
transform:rotate(35deg);
}
//画顶上的三角形
#star-five:before {
border-bottom:80px solid red;
border-left:30px solid transparent;
border-right:30px solid transparent;
position:absolute;
height:0;
width:0;
top: -45px;
left: -65px;
dispaly:block;
content:'';
transform:rotate(-35deg);
}
//画右上与左下的一个三角形
#star-five:after { 
position: absolute; 
display: block; 
color: red; 
top: 3px; 
left: -105px; 
width: 0px; 
height: 0px; 
border-right: 100px solid transparent; 
border-bottom: 70px solid red; 
border-left: 100px solid transparent;

-o-transform: rotate(-70deg);

}
