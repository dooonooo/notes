### hexo命令
简写
`hexo n` == `hexo new` #新建文章<br>
`hexo p` == `hexo publish` <br>
`hexo g` == `hexo generate` #生成<br>
`hexo s` == `hexo server` #启动服务预览<br>
`hexo d` == `hexo deploy` #部署<br>
监视文件变动
`hexo generate` #使使用hexo生成静态文件<br/>
`hexo generate --watch`  #监视文件变动
