**系统目录结构**
bin 可执行文件
boot 引导目录
dev 计算机上所有硬件设备
ect 几乎所有配置文件
home 所有用户家目录（root的home目录在根目录root目录）
<!--more-->
lib 库文件
lost+found
media 挂载目录
mut 挂载目录
opt 通常装大型文件
proc 系统的实时信息（存在内存中的虚拟目录）
root
sbin 可执行文件（只有root可以执行）
selinux
sys 系统底层硬件信息
tmp 临时目录（系统会自动删除）
usr 一般应用软件安装目录
var 保存经常变化的信息（典型的是log）
