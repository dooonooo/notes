CSS画图形

1.正方形

.square{
width:100px;
height:100px;
backgroud:red;}

2.长方形

width:150px;
height:100px;
background:red;}

3.圆形

//将正方形圆角放大到最大

border-bottom-left-radius:50px;
border-bottom-right-radius:50px;}

4.椭圆

5.三角形

前端优化的方法

  1. 减少http请求（合并压缩css、js、雪碧图（css-sprites）、cdn加速、data缓存、图片缓存）
  2. 前端模版
  3. 设置样式比较多的时候，设置className而不是直接操作样式
  4. 用innerHTML 代替DOM操作，减少DOM的操作次数，优化javascript性能
  5. 图片预加载，将样式放在顶部，将脚本放在底部加上时间戳。
  6. 避免在主体布局使用table。

尽量向前端优化，减少数据库操作，减少磁盘io，
在不影响功能体验上情况下，
能在浏览器执行的不要在服务端执行，
能在缓存服务器上直接返回的不要到应用服务器，
程序能直接得出结果的不要到外部取得，
本机能取得的数据不要到远程取
内存能取得的不要到磁盘取
缓存有的不要去数据库查询

以太坊的一些有用知识

以太命令

安装钱包（geth）--需要提前安装homebrew。
brew tap ethereum/ethereum
brew install ethereum

1.创建以太账户：geth account new
2.查询已创建的账户：geth account list
3.开启挖矿功能：geth --mine --minerthreads=4
(minerthreads=4是默认值，挖矿会占用电脑所有的计算资源，造成卡顿，可以改成2或者3)，等到generating DAG显示到达100%，就开始挖矿了

官方指南(英文)：

https://ethereum.gitbooks.io/frontier-guide/content/

在线查询帐户余额

https://etherchain.org/
输入你的地址，即可查询到你的余额（只有余额不为0的才可以查到）

Sublime笔记

转载并修改
原文链接： http://www.cnblogs.com/xiaohuochai/p/5073179.html

单词操作

1.选中当前单词 ctrl＋d
2.选中相同的所有单词 alt+f3

行操作

选中当前行 ctrl＋l

goto（ctrl＋p）

文件名 要打开的文件
:行 具体行(ctrl+g)
@名称 CSS选择器、HTMLid名及ID值、js事件及函数名(ctrl+r)
#关键字 具体的关键字(ctrl+;)

缩进

向前缩进tab ctrl+[
向后缩进 tab+shift、ctrl+]

注释

注释和取消注释单行 ctrl + /
注释和取消注释大段代码 ctrl + shift + /

折叠

折叠代码 ctrl+shift+[
展开代码 ctrl+shift+]

代码大小写

大写 ctrl+k ctrl+u
小写ctrl+k ctrl+l

其他

命令面板 ctrl+shift+p
侧边栏 ctrl+k ctrl+b
粘贴代码保持缩进格式 ctrl＋shiift＋v
闭合标签 alt＋,
选中光标的首尾标签 ctrl+shift+'

插件

package control安装
代码地址：https://packagecontrol.io/installation

emmet

[1] !+tab ->生成骨架结构
[2] #foo ->ID为'foo'
[3] .foo ->类名为'foo'
[4] + ->同级元素
[5] *3 ->生成3个元素
[6] w100 ->width:100px
[7] m5e ->margin:5em
[8] @f ->@font-face
[9] $ ->从1开始的递增值
[10] {} ->元素内容
