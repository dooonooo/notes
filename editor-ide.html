Sublime笔记

转载并修改
原文链接： http://www.cnblogs.com/xiaohuochai/p/5073179.html

单词操作

1.选中当前单词 ctrl＋d
2.选中相同的所有单词 alt+f3

行操作

选中当前行 ctrl＋l

goto（ctrl＋p）

文件名 要打开的文件
:行 具体行(ctrl+g)
@名称 CSS选择器、HTMLid名及ID值、js事件及函数名(ctrl+r)
#关键字 具体的关键字(ctrl+;)

缩进

向前缩进tab ctrl+[
向后缩进 tab+shift、ctrl+]

注释

注释和取消注释单行 ctrl + /
注释和取消注释大段代码 ctrl + shift + /

折叠

折叠代码 ctrl+shift+[
展开代码 ctrl+shift+]

代码大小写

大写 ctrl+k ctrl+u
小写ctrl+k ctrl+l

其他

命令面板 ctrl+shift+p
侧边栏 ctrl+k ctrl+b
粘贴代码保持缩进格式 ctrl＋shiift＋v
闭合标签 alt＋,
选中光标的首尾标签 ctrl+shift+'

插件

package control安装
代码地址：https://packagecontrol.io/installation

emmet

[1] !+tab ->生成骨架结构
[2] #foo ->ID为'foo'
[3] .foo ->类名为'foo'
[4] + ->同级元素
[5] *3 ->生成3个元素
[6] w100 ->width:100px
[7] m5e ->margin:5em
[8] @f ->@font-face
[9] $ ->从1开始的递增值
[10] {} ->元素内容
